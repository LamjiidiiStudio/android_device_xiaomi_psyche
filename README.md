The Xiaomi Mi 12X (codenamed _"psyche"_) is a flagship smartphone from Xiaomi.

It was released in December 2021.

## Device specifications

Basic   | Spec Sheet
-------:|:-------------------------
SoC     | Qualcomm SM8250-AC Snapdragon 870 5G (7 nm)
CPU     | Octa-core (1x3.2 GHz Kryo 585 & 3x2.42 GHz Kryo 585 & 4x1.80 GHz Kryo 585)
GPU     | Adreno 650
Memory  | 128GB 8GB RAM, 256GB 8GB RAM, 256GB 12GB RAM
Shipped Android Version | Android 11, MIUI 13 Got MIUI14 based on Android 13
Battery | Li-Po 4500 mAh, non-removable
Charging| 67W wired, PD3.0, QC4, 100% in 39 min (advertised)
Display | AMOLED, 68B colors, 120Hz, Dolby Vision, HDR10+, 1100 nits (peak), 1080 x 2400 pixels, 20:9 ratio (~419 ppi density)
M.Camera| 50 MP, f/1.9, 26mm (wide), 1/1.56", 1.0µm, PDAF, OIS, 13 MP, f/2.4, 12mm, 123˚ (ultrawide), 1/3.06", 1.12µm, 5 MP, f/2.4, 50mm (telephoto macro), AF, Dual-LED dual-tone flash, HDR, panorama
S.Camera| 32 MP, f/2.5, 26mm (wide), 0.7µm, HDR, panorama, 1080p@30/60fps, 720p@120fps, HDR
Sensors | Fingerprint (under display, optical), accelerometer, proximity, gyro, compass
